1-1-1 開發環境需求
1.開發時需要網頁伺服器（如：Apache）、資料庫（如：MySQL 或 MariaDB）、PHP 這些基本環境，理所當然的還
  需有個 XOOPS 網站。
2. XOOPS輕鬆架下載位址：https://campus-xoops.tn.edu.tw/modules/tad_uploader/index.php?of_cat_sn=11
   安裝之，那麼以上環境應該都有了。
3. windows 版的內含 Uniform Server 伺服器，解壓後，執行 UniController.exe 即可啟動網站。
4. 若無法裝在C磁碟，請按右鍵「以系統管理員身份執行」來安裝該exe檔
5. 預設會解壓縮到「C:\my_xoops」

   ⑶ 密碼:12345

   ⑸ 資料庫密碼:12345

6. 其實 Uniform Server 安裝在任何磁碟或目錄都可以，路徑不要有中文即可
7. 資料的帳號為 root，密碼為 2130669#33，
8. 日後若忘記資料庫密碼，可至 C:\my_xoops\htpasswd\mysql\passwd.txt 或
   C:\my_xoops\xoops_data\data\secure.php  查詢
9. 若要修改資料庫密碼，可點擊控制台的「MySQL→Change MySQL password」修改，然後要手動修改
   C:\my_xoops\xoops_data\data\secure.php 裡面的密碼。
10. 啟動資料庫 Start MySQL及網頁伺服器：Start Apache。
11. 若有跳出防火牆訊息，一律選「是」
12. 若無法啟動UniForm Server或者Apache無法啟動，請以管理員身份安裝 Visual Studio 2015~2019 Visual
    C++ 可轉散發套件（無法啟動輕鬆架時安裝）
